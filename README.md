# PRODIGY_DS_4

# Objective
Analyze and visualize sentiment patterns in social media data to understand public opinion and attitudes toward specific topics or brands.

# Dataset

twitter_training.csv
twitter_validation.csv

# Each dataset contains:

id: tweet identifier
entity: brand or topic mentioned
sentiment: Positive, Negative, Neutral, Irrelevant
text: tweet content

# Tools & Libraries

Python
Pandas
Matplotlib

# Steps Performed

Loaded training and validation datasets
Assigned appropriate column names
Combined both datasets into a single dataframe
Analyzed overall sentiment distribution
Identified top 5 entities based on frequency
Visualized sentiment patterns for these entities

# Visualizations

Bar chart showing overall sentiment distribution
Bar chart showing sentiment comparison across top 5 brands/topics

# Results

Negative sentiment is the most dominant overall
Some brands show strong negative sentiment, while others have balanced or positive opinions
Sentiment varies significantly across different entities, reflecting public attitude differences
